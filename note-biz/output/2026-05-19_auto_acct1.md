### STEP 1：Claude 3.5 Sonnetで「決裁者が唸るカリキュラム」を作る

まず、Claudeに構成を作らせます。
ここで重要なのは「汎用的な研修」にしないこと。相手の業界に特化させるのがコツです。

僕はいつもこのプロンプトをそのまま使ってます。

**【コピーして使ってください：研修構成案作成プロンプト】**

```markdown
# Role:
あなたは日本の中小企業向けにDX・AI導入コンサルティングを行うエキスパートです。
ITリテラシーがそこまで高くない社員でも「これなら使える！」と思わせる研修カリキュラムを作成してください。

# Target:
業界：[ここに業界名、例えば：建設業の事務部門]
悩み：[ここに悩み、例えば：見積書の作成に時間がかかる、電話対応が多すぎる]

# Mission:
以下の構成で、全5回の研修カリキュラム（各回60分想定）を作成してください。
1. なぜ今、この業界にAIが必要なのか？（危機感の醸成）
2. 専門知識不要！今日から使える魔法のフレーズ
3. 実践：[業界名]の具体的な業務をAIで自動化するワーク
4. セキュリティと著作権（社長が安心する内容）
5. 明日から何をすべきか？のアクションプラン

# Output Style:
- 各回について、話すべきトピックを5つずつ箇条書きで。
- Gamma（スライド生成AI）に流し込むための、マークダウン形式の「スライド原稿」として出力してください。
- 専門用語は極力排除し、中学生でもわかる言葉を使ってください。
```

これを送ると、一瞬で詳細なカリキュラムが出てきます。
「最初うまくいかなかった話」をまた一つすると、最初は「AIの仕組み」とか小難しいことを入れてたんですよね。でもそれ、現場の社員さんは1ミリも興味ないです。「これを使えば18時に帰れます」っていうメリットを強調するように書き換えさせてください。

### STEP 2：Gammaに流し込んで「一気に」スライド化する

構成案ができたら、次はGamma（https://gamma.app/）を使います。
ここ、操作を間違えるとデザインが崩れるので注意してください。

1.  Gammaにログインして、右上の **「Create new AI」** をクリック。
2.  **「Paste in text」** を選択します（ここ重要！「Generate」よりこっちの方が制御しやすいです）。
3.  Claudeが出力したマークダウン形式のテキストを、そのまま貼り付けます。
4.  右側のメニューで **「Presentation」** を選択し、スライドの枚数を指定します（だいたい20〜30枚くらいがリッチに見えておすすめ）。
5.  **「Continue」** を押すと、テーマ（デザイン）選択画面になります。僕はいつも「Professional」か「Corporate」系の落ち着いたネイビー系を選んでます。中小企業の社長はキラキラしたデザインより、信頼感がある方を好むんですよね。

これで、1分待つだけでスライドが出来上がります。
正直、初めてこれを見た時は「今までの僕のパワポ作成時間は何だったんだ…」ってまじで虚しくなりました。
