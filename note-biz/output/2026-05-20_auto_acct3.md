## STEP 1：ChatGPTで「泥臭い業務」を「100万円の価値」に変換するプロンプト

「自分には、AIで効率化したと言えるような大層な実績なんてない」

そう諦めるのは早い。
日々の泥臭い業務（週次のレポート作成、会議の議事録作成、顧客へのテンプレメール返信など）は、すべてChatGPTで効率化できるし、それを「年間削減コスト」に換算すれば、立派な実績になる。

まず、ChatGPT（GPT-4推奨）を開いて、以下のプロンプトをそのままコピペして実行する。


### 【コピペ用】業務効率化・実績定量化プロンプト

```text
# あなたの役割
優秀なビジネスアナリストおよび転職コンサルタント

# 前提条件
私は30代のビジネスパーソンで、転職活動に向けて「AIを活用して業務を効率化した実績」を数値化・言語化したいと考えています。私の泥臭い日常業務から、面接官に刺さる「コスト削減実績」と「生産性向上実績」を算出してください。

# 入力情報
1. 私の職種: [ここにあなたの職種を記入。例: 営業事務]
2. 効率化した（または効率化したい）業務: [例: 毎週金曜日の売上レポート作成と、会議用の議事録要約]
3. 手作業でかかっていた時間: [例: 毎週3時間]
4. ChatGPTやAIツールを使って短縮された（または想定される）時間: [例: 毎週30分に短縮]
5. 私の想定時給（または自社の平均時給）: [例: 2,000円]

# 出力フォーマット
以下の項目に沿って、具体的な数字を用いて出力してください。

■ 1. 年間削減時間とコスト削減額（算出式も明記）
■ 2. 職務経歴書にそのまま書ける「実績アピール文」（定量的な一文）
■ 3. 面接官のウケが良い「AI共生」をアピールするためのストーリー構成（Before → Action → After）
```


これを実行すると、例えば「毎週3時間のレポート作成を30分に短縮した」という地味な事実が、以下のようなインパクトのある実績に化ける。

*   **年間削減時間**: 120時間
*   **年間削減コスト**: 240,000円（自社メンバー5人に展開した場合、**年間120万円のコスト削減**）
*   **実績アピール文**: 「ChatGPTを活用したプロンプトテンプレートを自作し、属人化していた週次レポート作成業務を自動化。作業時間を83%削減（週3時間→30分）し、部署全体へ展開したことで年間120万円相当の人的コスト削減を主導」

## STEP 2：Notionで「面接官にURLを送るだけ」のポートフォリオを構築する

実績を数字にしたら、それをPDFの職務経歴書に文字で書くだけで終わらせてはいけない。
面接官が実際にアクセスして、クリックして動かせる「Notionポートフォリオ」としてWeb上に公開する。これが競合の転職希望者を一瞬でなぎ倒す武器になる。

今から、どこをクリックすればいいか、スマホやPCを用意して画面を見ながら進めてほしい。

### 1. Notionで新規ページを作成する
1. Notionを開き、左サイドバーにある**「ページを追加」**（＋マーク）をクリックする。
2. タイトルに「**【実績ポートフォリオ】〇〇 〇〇（あなたの名前）｜AI活用による業務効率化実績**」と入力する。
3. ページのアイコンに「💻」や「🚀」を設定し、カバー画像を設定して見栄えを整える。

### 2. 実績一覧のデータベースを作る
1. 本文領域で `/table` と半角で入力し、候補に出てくる**「テーブルビュー」**をクリックする。
2. データソースの選択画面で**「新規データベース」**をクリックする。
3. データベースの名前を「AI業務効率化・実績アーカイブ」にする。

### 3. プロパティ（列）を設定する
初期状態の列を、以下の5つのプロパティに変更する。

*   **名前（タイトル）**: 効率化した業務名（例：週次売上レポート作成の自動化）
*   **Before（数値型）**: 対策前の所要時間（単位：分）（例：180）
*   **After（数値型）**: 対策後の所要時間（単位：分）（例：30）
*   **削減率（数式型）**: 以下の数式を設定する。
    *   プロパティの種類を「数式（Formula）」に変更し、以下のコードを貼り付ける。
    ```javascript
    (prop("Before") - prop("After")) / prop("Before")
    ```
    *   数式の表示形式を「パーセント」に設定する。これで自動的に「83.3%」と表示される。
*   **成果物リンク（URL型）**: 実際に使ったChatGPTのプロンプト（Notionの別ページリンクでも可）や、自動化した成果物のサンプルPDFなどのリンク。

### 4. ページをWeb上に公開する
1. Notion画面の右上にある**「共有（Share）」**ボタンをクリックする。
2. **「ウェブで公開（Publish）」**タブをクリックする。
3. **「ネット上で公開（Publish to web）」**のトグルをONにする。
4. 生成された「ウェブ公開用リンク」をコピーする。

このURLを、履歴書や職務経歴書の「自己PR」や「特記事項」の欄に、**「※AI活用実績の詳細ポートフォリオはこちら（URL）」**と書いて貼り付けておく。

書類選考の時点で、採用担当者がこのリンクをクリックした瞬間、あなたの勝ちだ。他の候補者は全員、ただの「文字だけの履歴書」を出しているのだから。
