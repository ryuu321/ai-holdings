### STEP1：Vapiで「賢い受付嬢」の脳みそを作る

まずはVapi（vapi.ai）にログインします。
右上の「Create Assistant」をクリック。名前は「予約受付AI」とかでOKです。

ここで一番大事なのが「System Prompt」です。ここを適当に書くと、AIが変な敬語を使ったり、予約に関係ない世間話を始めたりして、クライアントからの信頼を失います。

僕が実際に整体院で使って、1発で「これすごいね！」と言わせたプロンプトをそのまま貼っときます。

**【Vapiシステムプロンプト：予約特化型】**
```text
# Role
あなたは「たくみ整体院」の優秀な電話受付スタッフです。
明るく丁寧な日本語で対応してください。

# Goal
1. 顧客の希望日時を聞き出す
2. Googleカレンダーの空き状況を確認する
3. 予約を確定させる
4. 顧客の名前と電話番号をヒアリングする

# Rules
- 敬語（丁寧語）を徹底してください。
- 予約時間は「◯月◯日の◯時」という形式で確認してください。
- 一度に多くの質問をせず、一つずつ確認してください。
- 空き状況を確認する際は「少々お待ちください」と伝え、ツールを実行してください。
- 予約が確定したら、最後に「お待ちしております」と締めてください。

# Specifics
- 施術メニュー：全身整体（60分）、骨盤矯正（40分）
- 営業時間：10:00〜20:00（最終受付19:00）
- 定休日：水曜日
```

これを「System Prompt」欄にコピペするだけで、それっぽい動きになります。

次に、声を選びます。ぶっちゃけ「Kyoko」とか「Nanami」あたりの落ち着いた声にしておけば間違いなかったりします。あまりに機械的な声だと、おじいちゃんおばあちゃんがびっくりして切っちゃうので。

### STEP2：Make.comを使ってGoogleカレンダーとガチ合体させる

ここが一番の難関で、僕が3日間つまずいたポイントです。
Vapi単体だと「予約を取るフリ」しかできないんですよね。実際にカレンダーを書き換えるには「Make.com」という連携ツールを使います。

やることはこれだけです。

1. **Makeで「Webhook」を作成**
   - Vapiから「予約したい」という信号を受け取るための入り口です。
   - Webhook URLをコピーして、Vapiの「Tools」に登録します。

2. **Googleカレンダーの「Search Events」を繋ぐ**
   - ここで、AIが「その時間、空いてる？」とカレンダーを見に行く設定にします。
   - 僕は最初、ここを飛ばして「空き状況無視で予約を入れる」という暴挙に出て、ダブルブッキングさせて先生にガチで怒られました。

3. **Googleカレンダーの「Create an Event」を繋ぐ**
   - 空いていたら予約を書き込みます。

Vapi側の「Tools」設定に書くJSONコードも、僕がデバッグしまくって完成させたやつを貼っときますね。

**【Vapi Tool Definition: check_availability】**
```json
{
  "type": "function",
  "function": {
    "name": "checkAvailability",
    "description": "Googleカレンダーの空き状況を確認します",
    "parameters": {
      "type": "object",
      "properties": {
        "date": { "type": "string", "description": "予約希望日 (YYYY-MM-DD)" },
        "time": { "type": "string", "description": "予約希望時間 (HH:mm)" }
      },
      "required": ["date", "time"]
    }
  }
}
```

これを設定して、Make側で「もし空いてたらOK、埋まってたらNG」を返すように組むんです。
正直、ここを自分でゼロから作ると1週間は溶けます。でも、この型があれば30分で終わったりします。

### 失敗から学んだ「運用上の注意点」

これ、まじで気をつけてほしいんですけど、AIは「想定外の言葉」に弱かったりします。
例えば、おばあちゃんが「あ〜、その日は孫の運動会があるから、雨だったら行けるんだけど……」みたいな曖昧なことを言うと、初期の僕の設定だとAIがフリーズしてました。

解決策は、Vapiの「First Message」を工夫することです。
「お電話ありがとうございます。予約専用のAI受付です。ご希望の日時を教えていただけますか？」と最初に「予約専用」であることを強調するのがコツだったりします。

あと、電話番号は「Twilio」で取得するのが一番安定します。050番号なら月数百円で持てるので、クライアントの負担も少なくて済みます。
