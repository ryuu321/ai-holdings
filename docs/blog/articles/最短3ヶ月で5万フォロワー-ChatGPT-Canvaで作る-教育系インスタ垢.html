#

1. ぶっちゃけ、最初は「地獄」でした

まず最初に、僕のダサい失敗談から話しておきますね。

半年前の僕は「インスタで稼げるらしい」という噂だけ聞いて、とりあえず「IT用語の解説アカウント」を作ったんです。エンジニアだし、知識はあるから余裕だろ、みたいな。

で、仕事が終わってから深夜の2時までかかって、Canvaでチマチマと文字を打って、画像を配置して、1枚の投稿を作りました。 「これ、まじで有益じゃん。バズるわ」 そう思って投稿したんですけど、翌朝の結果は「2いいね」。そのうち1つは僕のサブ垢です。

正直に言うと、これを3日続けた時点で心が折れました。「1投稿に4時間かけて、収益0円ってコスパ悪すぎだろ」って。副業月収8万なんて夢のまた夢だったんですよね。

そんな時、ふと「これ、ChatGPTに内容出させて、Canvaの機能で一気に流し込めないか？」と思いついたんです。それが全ての始まりでした。

2. 伸びるジャンルの「勝ちパターン」を見つけた話

「インスタはデザインが命」って思いがちなんですけど、教育系（豆知識やノウハウ系）に関しては、実は「構成」が9割だったりします。

僕が最初に失敗したのは「自分が書きたいこと」を書いていたからなんですよね。 今のインスタで伸びるのは、以下の3つの条件を満たすジャンルです。

1. 保存したくなる（後で見返したい知識） 2. 「え、そうなの？」という意外性がある 3. ターゲットが広い（英語、節約、便利ツール、暮らしの知恵など）

例えば、「Excelのショートカット」とか「100均で買える神アイテム」とか。これ、ChatGPTが得意な分野なんですよね。

僕は今、複数のアカウントを回してますが、先月の水曜日に新しく立ち上げた「雑学系アカウント」は、後述するやり方で運用して、開始2週間でフォロワーが1000人を超えました。まじで、自分で文章を考えないほうが伸びるっていう、皮肉な結果になったんです。

#

3. 【実録】ChatGPTから「バズるデータ」を抽出するプロンプト

ここからが具体的なやり方です。 ただChatGPTに「インスタのネタ考えて」って言っても、薄っぺらい回答しか来ません。

僕も最初は「英語のフレーズを5つ教えて」とか聞いてたんですけど、それだと「Hello」とか「Thank you」とか、誰でも知ってるようなことしか言わないんですよね。

そこで、僕はChatGPTを「編集者」として扱うようにしました。 実際に使っているプロンプトのイメージはこんな感じです。

> 「あなたはインスタでフォロワー10万人の教育系インフルエンサーです。20代の会社員が『明日から使える！』と感動するような、Excelの神ショートカットを30個、以下の表形式で出力してください。 > 項目：[投稿のタイトル], [紹介する機能], [操作方法], [得られるメリット] > 条件：10文字以内で、インパクトのある言葉を使ってください」

これを打つと、一瞬で30日分の投稿リストが出来上がります。 ぶっちゃけ、これを自分で調べてExcelにまとめるだけで3時間はかかりますよね。それが数秒。この時点で「勝ったな」と思いました。

このデータを「CSV形式」で保存するのが最大のポイントです。後でCanvaと連携させる時に、これが魔法の杖になります。

#

4. Canvaの「一括作成」で1ヶ月分を5分で作る

ここ、まじで一番大事なところです。 僕が月8万稼げるようになった最大の要因は、この「一括作成」機能を知ったことだったりします。

1. まず、Canvaで1枚だけ「型（テンプレート）」を作ります。 2. 左側のメニューから「アプリ」→「一括作成」を選択。 3. さっきChatGPTに出させたCSVファイルをアップロード。 4. デザイン上の文字の部分と、データの項目を紐づける（右クリックで「データの接続」を押すだけ）。 5. 「続行」ボタンを押す。

これをやると、一瞬で30枚、あるいは60枚の投稿画像が生成されるんです。 初めてこれを見た時、僕は職場のデスクで「まじか…」って声が出ました。今まで1枚ずつ文字を打ち込んでいた時間は何だったんだ、と。

正直に言うと、デザインの微調整は多少必要です。でも、4時間かかっていた作業が、ぶっちゃけ5分〜10分で終わるようになります。

5. マネタイズで月収20万を狙うロードマップ

投稿ができるようになったら、あとはどう稼ぐかですよね。 僕の場合、最初はアフィリエイトから始めました。

教育系アカウントなら、関連する本を紹介したり、資格スクールの資料請求に繋げたりするのが王道です。 僕の知り合いは、これと同じ手法で「暮らしの知恵」アカウントを運用して、楽天ルーム経由で月20万以上稼いでます。

ステップとしてはこんな感じです。 - 1ヶ月目： ChatGPTでネタ出し→Canvaで量産して、毎日1投稿（フィード＋リール）。まずはアルゴリズムに認識させる。 - 2ヶ月目： 保存数が多い投稿を分析。伸びた投稿の「類似ネタ」をChatGPTに再度出させる。 - 3ヶ月目： フォロワーが3000人を超えたあたりで、ストーリーズでアフィリエイトリンクを流す。

僕は今、3つのアカウントをこの方法で回していますが、1日の作業時間は合計でも30分くらいです。SIerの仕事が忙しくて、昼休みにスマホでポチポチ予約投稿するだけで、給料以外に8万入ってくる。これ、まじで精神的な余裕が違いますよ。

筋トレに行く時間も増えたし、最近はちょっといいプロテインを買えるようになりました（笑）。
