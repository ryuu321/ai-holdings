2. 「還元率30%」を狙うための具体的なカレンダー

僕が最初に取り組んだのは、寄付する「日」を固定することでした。

以前は「思い立った火曜日」とかにポチっていたんですが、これをやめました。僕が選んだのは楽天ふるさと納税ですが、徹底したのは以下の3つの条件が重なる日だけを狙うことです。

1. お買い物マラソンの期間中であること 2. 「0」か「5」のつく日であること 3. 勝ったら倍などのキャンペーンが重なっていること

例えば、先月の10日の水曜日。この日はお買い物マラソンの真っ最中で、なおかつ「0のつく日」でした。このタイミングを待って、あらかじめ「お気に入り」に入れておいた自治体に一気に寄付を入れます。

これ、意外と面倒そうに見えるんですが、慣れるとパズルみたいで楽しくなりました。

ここで大事なのが「買いまわり」の件数稼ぎです。ふるさと納税だけで10店舗回るのは寄付額的に無理（手取り22万だと上限が限られているので）ですが、普段使っている日用品（トイレットペーパーや洗剤など）をこの日にまとめて買うことで、ポイント倍率を無理やり10倍まで引き上げました。

これで、僕の場合は実質15%〜20%程度のポイント還元が常態化しました。寄付額が5万円だとしたら、それだけで7,500円〜10,000円分のポイントが戻ってくる計算です。これ、地方の昇給1年分より多いんですよね……。

4. 貯まったポイントを「新NISA」へ還流させるルート

ここからが本番です。貯まったポイントをどう使うか。 多くの人は、このポイントでまた別の買い物をしちゃいますよね。それだと、結局「消費」で終わってしまいます。

僕が構築したルートはこうです。

1. 楽天ポイント（通常ポイント）：すべて楽天証券での「投資信託」の買付に自動設定。 2. 期間限定ポイント：これは投資に回せないので、楽天ペイを使って「ドラッグストア」や「コンビニ」での生活費支払いに100%充当。 3. 浮いた現金：ポイントで支払った分、銀行口座に残った現金を、新NISAの買付余力（ネット銀行の自動振込機能）へ移動。

この「ポイントによる生活費の肩代わり」がめちゃくちゃ効きます。 例えば、先月はポイントで5,800円分の買い物をしました。本来なら財布から消えていたはずの5,800円が、そのまま特定口座やNISAの買付に回せるわけです。

数字で見ると、年間でこうなりました。 - ふるさと納税のポイント還元：約15,000円 - 返礼品（米・日用品）による生活費削減：約20,000円 - 合計：35,000円の軍資金

手取り22万円の僕にとって、年間3.5万円の追加投資ができるかどうかは、10年後の資産形成においてめちゃくちゃ大きいです。年利5%で運用できれば、これだけで将来大きな差になります。
