#

最初はまじで「嘘つきAI」に振り回されてました

今でこそ月8万とか言ってますけど、最初はこれ全然うまくいかなくて、むしろ大失敗してたんですよね。

去年の10月くらいだったかな。当時話題になってた普通のChatGPT（GPT-4）を使って、最新のワイヤレスイヤホンのレビュー記事を量産しようとしたんです。「最新の〇〇のスペックを教えて」ってプロンプトを打って、出てきた内容をそのままブログに貼ってたんですよ。

そしたら、読者から「この製品、その機能ついてませんよ」「価格が1万円も違います」ってコメントでボコボコに叩かれて。正直に言うと、冷や汗止まらなかったです。ChatGPTがもっともらしい嘘（ハルシネーションってやつですね）をついてるのに気づかずに、嘘の情報を垂れ流してたわけです。当然、アフィリエイト報酬は「0円」でした。

「やっぱりAIでブログなんて無理なのかな……」って、筋トレ中もずっと落ち込んでたんですよね。

#

11月20日の水曜日、転機が来ました

そんな時にリリースされたのが、ChatGPTの検索機能（ChatGPT Search）でした。

先月の11月20日、水曜日。いつものように残業して、ヘトヘトで帰ってきた夜に「これ、もしかしてリサーチに使えるんじゃね？」ってふと思ったんです。

試しに、その日発売されたばかりのガジェットについて「ネットから最新のレビューと価格、デメリットを検索して、比較表を作って」と指示してみたんですよ。そしたら、数秒で信頼性の高いソース元（公式サイトや有名レビューサイト）を引っ張ってきて、完璧なまとめを作ってくれたんですよね。

「あ、これだわ」って確信しました。

そこから試行錯誤して作ったのが、今回紹介する「10分で高品質なアフィリエイト記事を書き上げるプロンプト」です。これを使うようになってから、情報の正確性が爆上がりして、Googleからの評価も目に見えて変わったんですよね。

#

なぜ「ChatGPT Search」じゃないとダメなのか

ぶっちゃけ、普通のAI執筆だと「2023年までの情報」とかで止まってるじゃないですか。でもアフィリエイトで稼ぐには「今、この瞬間の情報」が命なんですよね。

- 最新のセール価格（Amazonブラックフライデーとか） - 発売されたばかりの新製品のスペック - リアルなSNSでの口コミ

これらを自動で、しかも「出典付き」で集めてくれるのが最強なんです。僕がやってるのは、AIが集めてきた情報を、プロンプトで「売れる構成」に流し込むだけ。正直、自分で考えてる時間はほとんどないです。

「[製品名]」について、最新のウェブ情報を検索して、アフィリエイト記事の構成案を作成してください。

# 命令書 1. 検索機能を使用して、以下の情報を正確に収集すること - 現在のAmazon/楽天での実売価格 - 主なスペック（3つ以上） - 良い口コミと悪い口コミ（SNSやレビューサイトから要約） - 競合製品との決定的な違い 2. 以下の「売れる構成」に従って執筆すること

# 構成案 - 導入: ターゲットが抱える悩み（例：〇〇選びで失敗したくない）への共感 - 結論: この製品は「買い」か？（結論を先に書く） - メリット・デメリット: 検索結果に基づいたリアルな声を含める - スペック比較表: markdown形式で出力 - どんな人におすすめか？: ターゲットを3パターンに分ける - まとめと購入のひと押し: 今買うべき理由（セール情報など）

プロンプトを打った後の「ひと手間」が収益を分ける

このプロンプトを使うと、一気に3,000文字くらいの構成が出てきます。 でも、ここでそのままブログにコピペするのはNGなんですよね。ここからが僕が月8万まで伸ばせたコツなんです。

僕がいつもやってるのは、「自分の体験談風の1行」を足すことです。

例えば、筋トレのサプリの記事なら「僕も実際に飲んでみたんですけど、ぶっちゃけ味は微妙でしたが、翌朝の感覚はまじで違いました」みたいな一文を、AIが作った文章の合間に入れるんです。

これだけで、Googleから「あ、これAIが書いただけのゴミ記事じゃないな」って判断されるようになるんですよね。SEとしてプログラムを組んでるから分かるんですけど、検索エンジンは「独自の視点」をめちゃくちゃ評価します。
