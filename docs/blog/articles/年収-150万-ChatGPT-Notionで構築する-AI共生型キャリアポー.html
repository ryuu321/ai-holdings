【コピペ用】業務効率化・実績定量化プロンプト

```text # あなたの役割 優秀なビジネスアナリストおよび転職コンサルタント

# 入力情報 1. 私の職種: [ここにあなたの職種を記入。例: 営業事務] 2. 効率化した（または効率化したい）業務: [例: 毎週金曜日の売上レポート作成と、会議用の議事録要約] 3. 手作業でかかっていた時間: [例: 毎週3時間] 4. ChatGPTやAIツールを使って短縮された（または想定される）時間: [例: 毎週30分に短縮] 5. 私の想定時給（または自社の平均時給）: [例: 2,000円]

# 出力フォーマット 以下の項目に沿って、具体的な数字を用いて出力してください。

■ 1. 年間削減時間とコスト削減額（算出式も明記） ■ 2. 職務経歴書にそのまま書ける「実績アピール文」（定量的な一文） ■ 3. 面接官のウケが良い「AI共生」をアピールするためのストーリー構成（Before → Action → After） ```

これを実行すると、例えば「毎週3時間のレポート作成を30分に短縮した」という地味な事実が、以下のようなインパクトのある実績に化ける。

* 年間削減時間: 120時間 * 年間削減コスト: 240,000円（自社メンバー5人に展開した場合、年間120万円のコスト削減） * 実績アピール文: 「ChatGPTを活用したプロンプトテンプレートを自作し、属人化していた週次レポート作成業務を自動化。作業時間を83%削減（週3時間→30分）し、部署全体へ展開したことで年間120万円相当の人的コスト削減を主導」

3. プロパティ（列）を設定する

初期状態の列を、以下の5つのプロパティに変更する。

* 名前（タイトル）: 効率化した業務名（例：週次売上レポート作成の自動化） * Before（数値型）: 対策前の所要時間（単位：分）（例：180） * After（数値型）: 対策後の所要時間（単位：分）（例：30） * 削減率（数式型）: 以下の数式を設定する。 * プロパティの種類を「数式（Formula）」に変更し、以下のコードを貼り付ける。 ```javascript (prop("Before") - prop("After")) / prop("Before") ``` * 数式の表示形式を「パーセント」に設定する。これで自動的に「83.3%」と表示される。 * 成果物リンク（URL型）: 実際に使ったChatGPTのプロンプト（Notionの別ページリンクでも可）や、自動化した成果物のサンプルPDFなどのリンク。
