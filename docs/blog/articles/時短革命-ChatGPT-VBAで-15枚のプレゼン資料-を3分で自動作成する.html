#

ステップ1：ChatGPTに「VBAコード」を吐き出させる魔法のプロンプト

ここが一番のキモです。 ただ「パワポ作って」と言うのではなく、以下のプロンプトをコピペして使ってください。 （※[ ] の部分は、作りたい資料の内容に合わせて書き換えてくださいね）

--- 【コピペ用プロンプト】

# 役割 あなたは超一流の経営コンサルタント兼、VBAエキスパートです。

# 目的 [ここにテーマを入れる。例：新卒採用向けの会社紹介資料] について、15枚構成のパワーポイント資料を自動作成するためのVBAコードを作成してください。

# 構成案の指示 - 1枚目：表紙 - 2枚目：アジェンダ - 3枚目〜14枚目：[具体的な内容。例：業界の現状、自社の強み、募集要項など]を詳細に展開 - 15枚目：サンクススライド

# VBAコードへの制約（重要！） 1. Microsoft PowerPoint Object Libraryへの参照設定が不要な「レイトバインディング」形式で書いてください。 2. 各スライドには「タイトル」と「本文（箇条書き）」を必ず含めてください。 3. 本文の内容は、専門的で説得力のある文章を1スライドあたり5行以上生成してください。 4. スライドのレイアウトは「ppLayoutText」を使用してください。 5. フォントサイズや色の設定は不要。まずは中身を完璧に流し込むことを優先してください。

---

これ、ポイントは「レイトバインディング」という言葉を入れているところです。 これを入れておかないと、マクロを実行するときに「参照設定」という面倒な作業が必要になって、初心者だとここで100%つまずきます。

#

ステップ2：出てきたコードをパワポに貼り付ける

ChatGPTがコードを書き終わったら（だいたい30秒くらいで終わります）、それをコピーします。

次に、パワーポイントを開いてください。 1. `Alt` + `F11` を押す（VBAのエディタが開きます） 2. 上のメニューの「挿入」→「標準モジュール」をクリック 3. 出てきた白い画面に、ChatGPTが書いたコードをペースト 4. `F5` キーを押す（実行！）

はい、これだけです。 画面の中で、スライドが次々と自動で生成されていく様子は、何度見ても快感ですよ。まじで。

#

実際に生成されたコードの例（貼っときます）

こんな感じのコードが出てきます。 （※これはあくまでイメージです。ChatGPTにその都度生成させてくださいね）

```vba Sub CreatePresentation() Dim pptApp As Object Dim pptPres As Object Dim slideIndex As Integer Set pptApp = CreateObject("PowerPoint.Application") Set pptPres = pptApp.Presentations.Add ' 1枚目：表紙 With pptPres.Slides.Add(1, 1) ' 1 = ppLayoutTitle .Shapes(1).TextFrame.TextRange.Text = "新卒採用向け会社紹介" .Shapes(2).TextFrame.TextRange.Text = "株式会社〇〇 IT事業部" End With ' (中略：ここに15枚分の処理がズラーっと入ります) pptApp.Visible = True End Sub ```

正直に言うと、たまにChatGPTがコードを途中で省略しちゃうことがあります。 その時は「続きを書いて」とか「15枚目まで全部省略せずにコードを書いてください」と言えばOKです。

ステップ3：デザインを「一瞬で」整える

「でも、これだけだと白黒でダサくないですか？」 って思った人、鋭いです。

でも大丈夫。今のパワーポイントには「デザイナー」という神機能があります。 生成されたスライドを選んで、右上の「デザイナー」ボタンを押すだけ。 AIが勝手にいい感じの画像やレイアウトを提案してくれるので、それをポチポチ選ぶだけで、3,000円で売れるレベルの資料が完成します。
