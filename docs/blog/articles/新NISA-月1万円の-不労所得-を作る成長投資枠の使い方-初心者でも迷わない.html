【新NISA】月1万円の「不労所得」を作る成長投資枠の使い方：初心者でも迷わない高配当株投資の完全ガイド

1. 「画面上の100万円」より「現金1,000円」が欲しかった

新NISAを始めたばかりの去年の今頃、私はちょっとした絶望感を味わっていました。 話題の「オルカン（eMAXIS Slim 全世界株式）」を毎月5万円積み立てていたんですが、3ヶ月経っても、半年経っても、私の手元にあるお金は1円も増えていない感覚だったんです。

もちろん、管理画面を見れば「＋12,400円」とか出ているんですよ。でも、それって売らないと使えないお金じゃないですか。

「老後のために今を我慢し続けるの、私には無理かも……」

そう思ってしまったのがきっかけでした。地方で一人暮らしをしていると、冬の灯油代が急に上がったり、車の車検で10万円飛んでいったりします。そんな時、画面上の評価益よりも、銀行口座に振り込まれる「チャリン」という配当金のほうが、どれだけ精神的な支えになるか。

そこから私は、新NISAの「成長投資枠」を使って、配当金をもらう「高配当株投資」を独学で勉強し始めました。

#

2. 最初は大失敗からのスタートでした

最初は、とにかく「利回りが高い銘柄」を適当に買っていました。 去年の6月、ちょうどボーナスが出たあとの水曜日でした。「利回り5.5%！めちゃくちゃお得じゃん！」と思って、ある地方銀行の株と、聞いたこともないような商社の株を、よく調べもせずに成長投資枠でドカンと買ったんです。

結果は散々でした。 買った1週間後には株価が急落。しかも、数ヶ月後には「減配（配当金が減ること）」の発表があって、利回りはガタ落ち。 「新NISAの非課税枠を無駄にした……」と、夜も眠れないくらい後悔しました。26歳の私にとって、数万円の損失は1ヶ月分の食費に相当しますから。

その時、ただ「利回りが高い」だけで選ぶのは、地方の「家賃が安すぎる事故物件」に飛び込むのと同じくらい危険なんだと痛感したんです。

#

3. 「月1万円の不労所得」を作るための具体的な計算

失敗を経て、私は数字を現実的に捉えることにしました。 「月1万円」の配当金をもらうには、年間で12万円の配当が必要です。

利回り4%の銘柄で運用するとしたら、必要な投資額は300万円。 「300万円なんて無理！」って思うかもしれませんが、新NISAの成長投資枠は年間240万円まで使えます。私は手取り22万の中から、固定費を月23,000円削って（格安SIMへの変更と、コンビニ飯をやめたのが大きかったです）、ボーナスも突っ込んで、2年かけてこの土台を作る計画を立てました。

実際、今の私のポートフォリオは平均利回り4.2%くらいに落ち着いています。 月1万円あれば、私の住んでいる地域の光熱費と水道代がほぼタダになる計算です。これ、意外と生活の質が変わるんですよね。

#

4. 地方在住の私が選ぶ「絶対に外さない」銘柄選びの3条件

私が失敗から学んだ、新NISAの成長投資枠で買うべき銘柄の条件は、以下の3つだけです。

① 「累進配当」を掲げている企業を選ぶ 「累進配当」っていうのは、「配当を減らしません、もしくは増やします」って宣言している企業のことです。三菱商事とか三井住友FGとか、超大手がこれを言っています。私みたいな初心者は、変に尖った銘柄より、こういう「絶対減らさないマン」についていくのが一番楽でした。

② 配当性向が40〜50%以下のもの 利益のうち、どれくらいを配当に回しているかの数字です。これが80%とか高い企業は、無理して配当を出しているので、将来的に「やっぱり無理でした！」って減らされるリスクが高いんです。余裕のある企業を選ぶのが、地方での穏やかな暮らしには合っています。

③ 自分の生活に関係がある業界 地方に住んでいると、どの銀行が強いか、どのインフラが欠かせないか、肌感覚でわかりますよね。私は自分が使っている銀行や、地元のガソリンスタンドと提携している商社など、ビジネスモデルが想像しやすいものから選ぶようにしました。

#

5. 具体的な買い方：一括購入は「毒」でした

これも失敗談なんですが、成長投資枠の240万円を一度に使い切ろうとするのは、初心者の私には心臓に悪すぎました。 今は「S株（単元未満株）」という仕組みを使って、1株単位で買うようにしています。

例えば、1株数千円の株を、毎週金曜日の夜に「今週もお疲れ様」という気持ちで1株ずつ買い足していくんです。これなら、株価が下がっても「安く買えたからラッキー」と思えます。

昨日の夜も、お風呂上がりにスマホで某通信株を2株だけ買いました。 スタバのフラペチーノ1回分を我慢すれば、将来ずっと配当を運んでくれる「金の卵」が手に入る。そう考えると、節約も全然苦じゃなくなりました。

#

6. 資産の「取り崩し」が怖い人へ

よく「老後は資産を4%ずつ取り崩せばいい」って言われますよね。 でも、26歳の私には、せっかく貯めた資産を削っていくなんて怖くて想像できませんでした。

高配当株投資のいいところは、「元本を触らなくていい」ことです。 木を切り倒して薪にするんじゃなくて、木に実った果実だけを収穫するイメージ。この感覚が、私のような心配性な性格にはぴったりでした。

新NISAの成長投資枠なら、その「果実（配当金）」にかかる約20%の税金もゼロ。 本来なら1万円の配当をもらっても、手元には8,000円しか残らないところが、丸々1万円入ってくる。この2,000円の差は、地方のランチ2回分です。めちゃくちゃ大きいですよね。

#

7. まとめと、明日からやること

「つみたて投資だけじゃ、なんだか実感がわかない」 そう思っているなら、成長投資枠で「1株だけ」高配当株を買ってみるのがおすすめです。

私がやってみて良かった流れをまとめます。

1. まずは固定費を月5,000円でもいいから削る（私はスマホを乗り換えただけで月6,000円浮きました） 2. 証券口座の「単元未満株」の設定を確認する 3. 「累進配当」をキーワードに、大手企業の株を1株買ってみる 4. 数ヶ月後、配当金が数十円でも入ってくる喜びを味わう

最初は数十円、数百円かもしれません。 でも、そのお金はあなたが働いて稼いだお金ではなく、あなたのお金が働いて連れてきてくれたお金です。

私は今、月1万円の配当金を目指して、成長投資枠をコツコツ埋めている最中です。 地方住み、手取り22万、奨学金あり。 そんな私でも、新NISAを正しく使えば、少しずつ「お金に困らない未来」を作れている実感があります。
