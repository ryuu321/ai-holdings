【公開】ビズリーチのプラチナスカウトを3倍に増やした『プロフィール入力』15の鉄則

ビズリーチに登録したのに、来るのは「誰にでも送ってるだろ」っていう定型文のスカウトばかり。年収アップどころか、今の会社より条件が悪い案件が届いて、スマホを投げ出したくなっていませんか。

ぼくも3年前、同じ状況でした。当時は年収350万。現状を変えたくて登録したのに、1週間待ってもプラチナスカウトはゼロ。届くのは「とりあえず面談しませんか？」という質の低い連絡だけ。でも、ある「プロフィールの書き方」に気づいてから、状況は一変しました。

結果として、2回目の転職で年収を150万引き上げ、6.5M（650万円）の提示を勝ち取ることができました。

この記事では、ぼくが泥をすすりながら見つけた「採用担当者の検索画面で1位に表示され、かつ一瞬で指名されるプロフィール」の作り方を、15の鉄則としてまとめました。これ、実際にぼくが使った一言や、エージェントに「これ書かれるとクリックしちゃうんだよね」と言われた生の情報だけを詰め込んでいます。

続きに全部書きました。

#

ぼくの失敗談から先に言います

まず、ぼくが一番最初にビズリーチで大失敗した時の話をします。 26歳の時、新卒で入った会社を2年で辞めた直後です。当時のぼくは「実績があれば、適当に職歴を埋めるだけでスカウトが来る」と勘違いしていました。

職務要約には「営業として売上に貢献しました」とだけ書き、職務詳細も会社から渡された募集要項をコピペしたような内容。その結果どうなったか。

1ヶ月待っても、まともなプラチナスカウトは1通も来ませんでした。 たまに来るのは、誰でも受かるような大量採用の兵隊案件。年収は現状維持か、下がるものばかり。あの時、仕事終わりのベローチェで自分のプロフィールを読み返して、「これ、自分でもスカウト送らないな」と絶望したのを今でも鮮明に覚えています。

そこから、転職エージェントの友人に土下座して、彼らが使っている「企業側の管理画面」を見せてもらいました。そこで衝撃を受けました。

採用担当者は、あなたのプロフィールを「読んで」いません。 「検索」して「流し見」しているだけです。

AIによるマッチングアルゴリズムと、人間の担当者の「直感」の両方をハックしない限り、プラチナスカウトの通知は鳴りません。 ぼくが年収を150万アップさせた時に実践した、具体的すぎる15の鉄則を公開します。

#

鉄則1：更新ボタンは「平日の午前8時30分」に押す

これ、意外と誰もやっていないんですが、一番即効性があります。 ビズリーチの検索画面では「更新日が新しい順」に候補者が並びます。採用担当者がデスクについて、コーヒーを飲みながら最初にログインするのが9時前後。その瞬間にあなたのプロフィールが一番上にいなければなりません。

ぼくは転職活動中、毎朝8時30分に職務要約の末尾に「。」を付け足したり消したりして保存し直していました。これだけで、先月の水曜日には1日で3通のプラチナスカウトが届きました。

#

鉄則2：職種名は「一般的な名称」＋「専門キーワード」で固める

例えば「営業」だけでは検索に引っかかりません。 ぼくが実際に使った表記はこれです。 「法人営業（IT SaaS / エンタープライズ領域 / 新規開拓メイン）」

AIは括弧内のキーワードを拾います。自分がやりたい仕事の求人票を5枚並べて、そこに出てくる単語をそのまま職種名に盛り込む。これが検索順位を上げる最短ルートです。

#

鉄則3：職務要約の1行目に「実績の数字」を叩き込む

「私の強みは粘り強さです」なんて書いた瞬間にブラウザバックされます。 ぼくが書き直した1行目はこれです。 「【達成率140%】入社2年目で年間売上1.2億円を達成し、全社300名の中でMVPを受賞」

担当者は最初の3行しか見ていません。そこに「通貨単位（円）」と「％」と「順位」を全部入れます。

#

鉄則4：「特殊な社内用語」をすべて排除する

あなたの会社でしか通じないプロジェクト名や役職名は、AIにとってゴミと同じです。 「第2営業部・特販チーム・リーダー」ではなく、「大手製造業向け・直販営業・チームマネジメント（5名）」と書く。誰にでもわかる言葉に翻訳することで、他業界からのプラチナスカウトを呼び込みます。

#

鉄則5：使用ツールを「これでもか」と羅列する

今の転職市場、特に高年収帯では「何ができるか」以上に「何を使ってきたか」が見られます。 ぼくは「スキル」の欄に、以下のツールを全て書きました。 ・Salesforce (CRM) ・Slack / Zoom ・Notion ・Google Workspace ・Tableau これらが並んでいるだけで、「ITリテラシーの教育コストがかからない人材」だとAIが判断します。

#

鉄則6：プロジェクトの「予算規模」を明記する

年収500万以上の壁を超えるには、扱ってきた金の大きさをアピールする必要があります。 「大規模なシステム改修を担当」ではなく、「予算5,000万円、期間8ヶ月の基幹システム刷新プロジェクトのPMOを担当」と書く。数字の桁数が増えるほど、プラチナスカウトの年収提示額も上がります。

#

鉄則7：「自己PR」は箇条書きにする

文章で書かれた自己PRを、忙しい人事は読みません。 ・【課題解決力】〇〇の課題に対し、〇〇を導入し、コスト20%削減 ・【適応力】新規事業立ち上げにおいて、3ヶ月で0から1の体制構築 このように、見出しをつけて一瞬でメリットが伝わるように構成します。

#

鉄則8：退職理由は「達成感」で締める

「不満があって辞めたい」と思われては負けです。 ぼくが面接でもプロフィールでも使った鉄板のフレーズはこれです。 「現職にて〇〇の目標を達成し、一定の成果を出し切ったと感じています。次は〇〇の領域で自分のスキルを試したいと考え、転職を検討しています。」 これなら「逃げの転職」ではなく「攻めの転職」に見えます。

#

鉄則9：希望年収は「現状＋100万」を最低ラインにする

謙遜して「今の年収と同じ」に設定すると、今の年収以下のスカウトしか来ません。 ぼくは年収500万の時に、あえて「650万以上」と設定しました。すると、本当にその金額を出せる企業からしか連絡が来なくなります。スカウトの数は減りますが、質が劇的に上がります。

#

鉄則10：「英語力」がなくてもTOEICの点数を書く

もし600点以上持っているなら、たとえ仕事で使っていなくても必ず書きます。 外資系企業の採用担当者は、フィルターで「TOEIC 600以上」とチェックを入れることが多いからです。これで足切りを突破できます。

#

鉄則11：顔写真は「プロ」に撮らせたものを使う

自撮りや、証明写真機で撮った写真は今すぐ消してください。 ぼくは3,000円払ってスタジオで撮ってもらいました。これだけで「デキる男」感が演出できます。視覚情報は、ロジックを凌駕します。

#

鉄則12：スカウトへの返信は「15分以内」を心がける

これ、裏技です。返信速度が速い候補者は、ビズリーチ側で「転職意欲が高い」とフラグが立ちます。 ぼくは仕事中も通知をオンにして、定型文でもいいから即レスしていました。すると、さらに別の企業からのスカウトが増えるという好循環に入りました。

#

鉄則13：マネジメント経験は「人数」と「役割」を具体化する

役職がなくても「後輩2名の教育担当」なら立派なマネジメント経験です。 「メンターとして新人2名を指導し、早期立ち上げに貢献」と書くだけで、リーダー候補としてのスカウトが届くようになります。

#

鉄則14：「やりたくないこと」を逆説的に書く

「テレアポはやりたくありません」と書くのではなく、「戦略的なインバウンドセールスに特化した環境で、LTV向上に寄与したい」と書きます。これで、質の低い営業職スカウトをフィルタリングできます。

#

鉄則15：最後に「追伸」を職務要約に入れる

これ、ぼくが編み出したテクニックの中で一番効果がありました。 職務要約の最後にこう書きます。 「※プラチナスカウトをいただいた方には、24時間以内に必ず返信いたします。」 これを見た採用担当者は「この人に送れば確実に反応がもらえる」と思い、送信ボタンを押す心理的ハードルが下がります。

まとめと次にやること

ここまで15の鉄則を話してきましたが、結局、動くか動かないかです。 ぼくは最初の転職で失敗し、年収が上がらないどころか、残業代も出ないブラック企業に捕まりかけました。あの時、プライドを捨ててプロフィールを1から書き直したからこそ、今の1.5Mアップした生活があります。

まずは今すぐビズリーチを開いて、鉄則1の「更新」と、鉄則3の「1行目の数字」だけ変えてみてください。

明日の朝、あなたのスマホの通知欄がプラチナスカウトで埋まっているはずです。

次にやるべきこと： 1. 今すぐプロフィール画面を開く 2. 職務要約の1行目を「数字」に書き換える 3. 明日の朝8:30に「保存」ボタンを押す

これだけで、あなたの市場価値は正しく評価され始めます。応援しています。
