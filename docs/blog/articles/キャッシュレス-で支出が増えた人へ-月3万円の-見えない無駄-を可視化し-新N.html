1. 「便利さ」という名のブラックホールに落ちていた話

まず、僕の恥ずかしい失敗談から聞いてください。

去年の10月、仕事帰りに地元のドラッグストアに寄った時のことです。手取り22万円、そこから奨学金の返済で毎月15,000円が消えていく僕にとって、財布の紐は固いはずでした。

でも、スマホをかざすだけで決済が終わる「キャッシュレス」は、僕の金銭感覚を完全にバグらせていたんです。

「ポイントが貯まるからお得だし」 「財布を出さなくていいから時短になるし」

そんな言い訳をしながら、仕事のストレス解消に新作のコンビニスイーツを買い、なんとなく目についた高い入浴剤をカゴに入れ、気がつけば1回の買い物で3,000円。それが週に3回。

月末、クレジットカードの請求額を見て、膝から崩れ落ちました。請求額「148,000円」。家賃5万円、奨学金1.5万円、光熱費や通信費を合わせると、もう食費すら残らないレベルでした。

「あれ？ 贅沢した記憶、全くないんだけどな…」

これがキャッシュレスの怖さでした。現金の重みがないから、脳が「お金を使っている」と認識してくれないんです。このままじゃ一生、投資なんて無理。そう確信した瞬間でした。

2. ルール①：クレカを封印し「デビットカード」を主役にする

キャッシュレスで失敗する最大の理由は「支払いと引き落としのタイムラグ」です。 1ヶ月後に請求が来るから、今いくら使えるのかが直感的にわからない。

そこで僕が最初にしたのは、メインの決済をクレジットカードから「銀行デビットカード」に変えることでした。

これが、意外と効果絶大でした。 デビットカードは、使った瞬間に銀行口座からお金が引き落とされます。スマホの通知に「2,480円引き落とされました。残高は42,100円です」と即座に出るように設定してみたんですが、これが効くんです。

「あ、今の買い物で残高が4万切っちゃう…」

この「残高が減る痛み」をリアルタイムで感じることで、無意識のついで買いが激減しました。具体的には、この設定に変えただけで、翌月のコンビニ利用額が月12,000円から3,000円まで落ちました。差額の9,000円、これだけで新NISAの積立額を増やせますよね。

地方だと車移動が基本なので、ガソリン代だけはクレカ（ポイント還元のため）にしていますが、それ以外のスーパーやドラッグストアの買い物は、すべて「即時引き落とし」のルールに徹底しました。

#

3. ルール②：PayPayなどのQR決済は「月1回・定額チャージ」のみ

次にメスを入れたのが、PayPayや楽天ペイなどのQR決済です。 以前の僕は「オートチャージ」を設定していました。残高が減ったら勝手に銀行からチャージされる。これ、家計管理においては「底なし沼」と同じでした。

去年の12月、忘年会シーズンにこれをやっていたら、いつの間にか月5万円もPayPayで使っていて震えました。

そこで、以下の運用に変えてみたんですが、これが驚くほど管理を楽にしてくれました。

- 毎月1日に、その月の「お小遣い」として20,000円だけチャージする。 - オートチャージは設定オフ。 - 足りなくなったら「今月はもう遊べない」と自分に言い聞かせる。

これ、最初は「不便そうだな」と思ったんですが、意外とそうでもありませんでした。 むしろ「今週はあと5,000円残ってるから、週末にちょっといいランチ行けるな」と、ゲーム感覚で予算をやりくりする楽しさが生まれたんです。

この「定額チャージ制」にしたことで、使途不明金だった月15,000円ほどの「見えない無駄」が完全に可視化されました。

4. ルール③：新NISAの入金分を「給料日に別口座へ強制送金」する

最後にして最強のルールが、支出を抑えるのではなく「先に投資分を確保する」ことです。 手取り22万円の中から、僕は現在5万円を新NISAに回しています。

「そんなの無理だよ」と思うかもしれませんが、仕組みを作れば意外とできてしまいました。

僕がやったのは、給料日の25日に、以下のフローを自動化したこと。

1. 給料が入る（三菱UFJ銀行） 2. 自動振込機能で、5万円を「投資用口座（楽天銀行）」へ飛ばす 3. 楽天証券で、その5万円を新NISA（eMAXIS Slim 全世界株式）の積立に自動設定

このポイントは、自分の意志を介在させないことです。 「今月余ったら投資しよう」と思っているうちは、絶対に余りません。人間だもの。

僕は以前、手元にお金があると「今月は頑張ったから」と、地方民の聖地であるイオンモールで余計な服を買ってしまう癖がありました。でも、先に5万円を抜いてしまえば、残された17万円（そこから家賃や固定費を引いた額）で生活するしかなくなります。

この「強制隔離」を始めてから、家計管理のストレスが激減しました。 だって、投資分はもう確保されているから、残りの範囲内なら何に使っても「将来への罪悪感」がないんです。
