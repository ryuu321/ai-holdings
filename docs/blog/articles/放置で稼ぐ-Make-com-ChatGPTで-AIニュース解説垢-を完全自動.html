【放置で稼ぐ】Make.com×ChatGPTで「AIニュース解説垢」を完全自動化し、30分で1ヶ月分の投稿を生成する全手順

どうも、たくみです。都内のSIerで社畜やりつつ、副業で月8万くらい稼いでる28歳です。

ぶっちゃけ、平日は22時過ぎまで残業とかザラなんですよね。家に帰ってから「よし、副業のSNS投稿作るぞ！」なんて気力、まじで1ミリも残ってなかったりします。筋トレも行きたいし、ゲームもしたいじゃないですか。

最初は僕も、毎日手動で海外のAIニュースを調べて、必死に要約して、Canvaで画像作って……ってやってたんですけど、3日で挫折しました。先月の水曜日の夜、PCの前で寝落ちして、気づいたら朝の4時だった時は「あ、これ無理だわ」って正直に思いましたね。

でも、そこから「全部自動化しちゃえば良くない？」って気づいて、Make.comとChatGPTを繋ぎまくった結果、30分の作業で1ヶ月分の投稿が勝手に出来上がる仕組みが完成したんです。これのおかげで、今は寝てる間もAIが勝手に働いてくれてます。

忙しくて副業の時間がないって人にこそ、この「サボりながら稼ぐ仕組み」を知ってほしいなと思って、僕が泥臭く試行錯誤した設計図を全部まとめました。

続きに全部書きました。

📊 AIが毎日6つのボットの投資シグナルを無料配信中 → https://t.me/+yUiqVJi2uNFiOTA1

正直に言うと、最初は「自動化なんてエンジニアの俺なら余裕っしょ」って舐めてたんですよね。

でも、最初に組んだ仕組みはまじでゴミでした。Googleニュースから情報を引っ張ってきて、ChatGPTに「要約して」って投げただけだったんですけど、出来上がった文章が「〜であります」「〜と言えるでしょう」みたいな、いかにもAIが書きました！っていうガチガチの硬い文章になっちゃって。あんなのSNSで誰も読まないですよね。

しかも、画像生成のプロンプトも適当だったから、ニュースの内容と全然関係ない、なんかサイバーパンクな街の画像が毎回生成されたりして。フォロワーからも「これ何のアカウント？」って思われてたはずです。ぶっちゃけ、最初の2週間は収益どころかフォロワーも10人も増えなくて、API代の2,000円くらいが溶けただけでした。

そこから「どうすれば人間味を出せるか」「どうすればクリックされるか」を筋トレしながら必死に考えて、ようやくたどり着いたのがこの設計図です。

#

1. 全体像：4つのツールを「連結」させる

僕が最終的に落ち着いた構成はこれです。

1. RSS Feed / Google News: 海外の最新AIニュースを拾う 2. Make.com: 全体の司令塔（ハブ） 3. ChatGPT (OpenAI API): ニュースの選別・要約・SNS向け変換 4. DALL-E 3 / Canva: アイキャッチ画像の自動生成 5. Googleスプレッドシート: 投稿内容のストックと予約管理

これをMake.comで繋ぐんですけど、ポイントは「いきなり投稿しないこと」なんですよね。一旦スプレッドシートに溜めるのがコツです。

#

2. 【失敗談】なぜ「即投稿」はダメなのか

これ、僕がやらかしたから言えるんですけど、自動化してそのままX（Twitter）とかに流すと、たまに翻訳ミスで意味不明なニュースが混ざるんですよ。

先月の20日くらいに、全然関係ない「海外のセレブのゴシップ」をAIが「最新AIトレンド」として要約しちゃって、そのまま投稿されたことがありました。まじで恥ずかしかったです。

だから、今のフローはこうなってます。

- STEP 1: Make.comが1時間に1回、海外サイトを見に行く。 - STEP 2: ChatGPTが「これはAIに関する重要なニュースか？」を判定。 - STEP 3: 重要なら、SNSでバズりやすい「タイトル」と「解説文」を生成。 - STEP 4: 画像も生成して、一旦スプレッドシートに1行追加。

これで、週末の30分くらいでスプレッドシートをサッと確認して、変なのがないかチェックするだけでOK。あとはBotが予約投稿してくれる仕組みです。

#

3. ChatGPTに「中の人」を演じさせるプロンプト

ここが一番大事なノウハウなんですけど、ChatGPTに普通に頼んじゃダメなんですよね。 僕が実際に使ってるプロンプトの一部を貼っときます。

> 「あなたは20代後半のIT企業勤務の男性です。友達に教えるような口調で、以下のニュースを3つのポイントで解説してください。 > 条件： > ・『〜なんですよね』『〜だったりします』を語尾に使う > ・専門用語は中学生でもわかるように噛み砕く > ・最初に『これ、まじでヤバいです』みたいな惹きの一言を入れる」

これを入れるだけで、AI臭さが一気に消えます。正直に言うと、僕が自分で書くより読みやすい文章が出てきたりして、ちょっと複雑な気持ちになったりもします（笑）。

#

4. 実際のMake.comの設定画面

（ここにMake.comのモジュールが繋がっているスクショのイメージ）

設定のコツは、OpenAIのモデルを「GPT-4o」にすること。 最初はケチって「GPT-3.5」を使ってたんですけど、やっぱり要約の質が全然違います。月8万稼げるようになったのは、ここをケチるのをやめてからですね。API代で月3,000円くらいかかりますけど、リターンの方が圧倒的に大きいです。

画像生成についても、DALL-E 3に「16:9のサイズで、テック系のおしゃれなアイキャッチを作って。文字は入れないで」って指示を固定してます。文字を入れると、AIはまだスペルミスが多いので、画像はイメージ図として割り切るのが正解だったりします。

#

5. これで月8万稼ぐまでのステップ

「で、どうやってマネタイズするの？」って話ですけど、僕は主に2つやってます。

1. アフィリエイト: ニュースの文末に「このAIツール、ここで使えます」ってリンクを貼る。 2. Tips/noteへの誘導: 詳しく知りたい人向けに、解説記事を売る。

これを完全自動で、1日3回投稿するようにしてから、フォロワーが勝手に増えて、リンクを踏んでくれる人が激増しました。

先月はアフィリエイトだけで5万、noteの販売で3万くらい。合計8万ちょっと。 SIerの給料以外にこれだけ入ってくると、まじで心の余裕が違います。スーパーで値段見ずに肉買えるようになったり、欲しかったゲーミングチェアをポチれたり。

#

まとめと次にやること

ここまで読んでくれた人に、正直に言います。 「設定めんどくさそう」って思いましたよね？

確かに最初はちょっとだけ面倒です。僕も最初の土日は、Make.comの繋ぎ方がわからなくて「エラー：400」とか出て、PC投げ捨てそうになりました。

でも、一度作っちゃえば、あとは「不労所得」に近い状態になります。

次にやるべきことリスト： 1. Make.comのアカウントを作る（無料プランでOK） 2. OpenAIのAPIキーを取得する（5ドル分くらいチャージする） 3. 自分の好きなジャンル（AI、美容、投資なんでもOK）のRSSサイトを探す 4. 僕が書いたプロンプトをコピペして繋いでみる

最初は失敗してもいいんです。僕なんて、自動投稿ボットが暴走して同じ内容を100件連続でツイートしちゃって、アカウント凍結寸前までいったこともありますから（笑）。

でも、そこで「あ、これエラーハンドリング甘かったな」って修正していくだけ。 その積み重ねが、月8万っていう数字に繋がったんですよね。

まずは、今日中にMake.comを触ってみることから始めてみてください。まじで世界変わりますよ。
