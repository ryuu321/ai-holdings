# /ventures — 事業候補リストを表示する

`secretary/knowledge/ventures.md` を読んで、事業候補の現状を報告する。

## 手順

1. `secretary/knowledge/ventures.md` を読む
2. 以下の形式で出力する

## 出力フォーマット

```
事業候補リスト — {最終更新日}

【着手中】
  - {事業名}（登録: {日付}、登録元: {手動/リサーチ}）

【候補】
  手動登録: {件数}件
  リサーチ検出: {件数}件
  合計: {件数}件

【直近追加（リサーチ検出）】
  - {事業名}（スコア: {score}/40、{日付}）
  - ...

【ステータス別サマリー】
  候補: X件 / 着手中: X件 / 運用中: X件 / 凍結: X件 / 却下: X件

何か着手しますか？
```

## 注意
- ventures.md にないことは推測しない
- リサーチ検出分は登録日とスコアを必ず表示する
