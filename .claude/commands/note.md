# /note — note記事を1本生成する

`note-biz/` の各エージェントを順番に動かして、投稿可能な記事を1本生成する。

## 実行手順

1. `note-biz/researcher/CLAUDE.md` を読む
2. バックログからテーマを1つ選ぶ（推奨度★★★優先・前号との繋がりを考慮）
3. `note-biz/skills/article-generator.md` のフォーマットに従って記事を生成する
4. `note-biz/writer/CLAUDE.md` の文体ルール・感情アークに従って仕上げる
5. `note-biz/seo/CLAUDE.md` に従って最適化レポートを出力する
6. 記事本文を `note-biz/output/YYYY-MM-DD_article.md` に保存する

## 出力形式

```
=== 記事生成完了 ===

【テーマ】{選んだテーマ}
【タイトル】{最終タイトル}
【文字数】{N}字（無料: {N}字 / 有料: {N}字）

--- 記事本文 ---
{記事全文}
---

=== SEO最適化レポート ===
{note-biz/seo/CLAUDE.mdのアウトプット形式に従ったレポート}

=== 投稿チェックリスト ===
- [ ] タイトルが32字以内か
- [ ] 無料部分が300字以内か
- [ ] ハッシュタグ #{タグ1} #{タグ2} #{タグ3} #{タグ4} をセットする
- [ ] 投稿タイミング: {推奨曜日・時間帯}
- [ ] マガジン「チビの哲学」に追加する
```

## 引数（省略可）

```
/note テーマを指定して生成したい場合はここに書く
例: /note 嫉妬が止まらない男の話
```

引数がない場合はバックログから自動選定する。
