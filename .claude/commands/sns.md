# /sns [プラットフォーム] — SNS投稿を生成する

指定プラットフォームの投稿コンテンツを生成する。

## 使い方
```
/sns instagram        → Instagram投稿を3本生成
/sns instagram 分析系 → 分析系投稿を1本生成
/sns x               → X投稿を3本生成（フェーズ2以降）
```

## 実行手順

### instagram の場合
1. `sns-ops/instagram/CLAUDE.md` を読む
2. コンテンツバックログから未投稿ネタを選ぶ
3. 型A/B/Cに従って投稿を生成する
4. ハッシュタグ・投稿タイミングも出力する
5. `sns-ops/output/YYYY-MM-DD_instagram.md` に保存する

### x の場合
1. `sns-ops/x/CLAUDE.md` を読む
2. 型A/B/Cに従って投稿を生成する
3. `sns-ops/output/YYYY-MM-DD_tweets.md` に保存する

## 出力形式
```
=== Instagram投稿 生成完了 ===

### 投稿① （型A：分析系）
【サムネイルテキスト】
...

【キャプション】
...

【ハッシュタグ】
#サロン集客 #ネイルサロン集客 #個人サロン #サロン経営 #Instagram運用 #美容集客

【投稿タイミング】火曜 20:00推奨

---
（以下、投稿②③も同様）

=== 投稿チェックリスト ===
- [ ] サムネイル画像を作成する（Canva等）
- [ ] キャプションをコピーしてInstagramに貼る
- [ ] ハッシュタグをセットする
- [ ] 予約投稿をセットする
```
